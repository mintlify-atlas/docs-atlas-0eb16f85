Milkdown is a lightweight, plugin-driven WYSIWYG markdown editor framework. Built on top of [ProseMirror](https://prosemirror.net/) and [remark](https://github.com/remarkjs/remark), it lets you compose rich editing experiences from focused, composable plugins — or get started instantly with the opinionated **Crepe** editor.

## Why Milkdown?

Milkdown gives you full control over the editing experience without forcing you to start from scratch. Every feature — from GFM tables to slash commands to collaborative editing — is an independent plugin you can include, configure, or replace.

## Get started

<Steps>
  <Step title="Install the packages">
    Install `@milkdown/kit` for the full editor toolkit, or individual packages for a minimal setup.

    ```bash
    npm install @milkdown/kit
    ```
  </Step>
  <Step title="Create an editor">
    Use the `Editor` class to compose an editor with the plugins you need.

    ```typescript
    import { Editor } from '@milkdown/kit/core'
    import { commonmark } from '@milkdown/kit/preset/commonmark'

    const editor = await Editor.make()
      .config((ctx) => {
        // configure root element, default value, etc.
      })
      .use(commonmark)
      .create()
    ```
  </Step>
  <Step title="Or use Crepe for a ready-made editor">
    The `Crepe` class bundles all popular features with sensible defaults.

    ```typescript
    import { Crepe } from '@milkdown/crepe'
    import '@milkdown/crepe/theme/common/style.css'
    import '@milkdown/crepe/theme/classic.css'

    const crepe = new Crepe({ root: document.getElementById('editor') })
    await crepe.create()
    ```
  </Step>
  <Step title="Integrate with React or Vue">
    Use the official framework packages for seamless component integration.

    ```tsx
    import { Milkdown, MilkdownProvider, useEditor } from '@milkdown/react'
    import { commonmark } from '@milkdown/kit/preset/commonmark'

    export function MyEditor() {
      useEditor((root) =>
        Editor.make().config(ctx => { /* ... */ }).use(commonmark)
      )
      return <Milkdown />
    }
    ```
  </Step>
</Steps>
